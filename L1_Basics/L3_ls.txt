ls



	See the hidden files ?
		
		ls -a


	Sort the files?
		
		ls -c  (Sort By Last Modification)

		ls -l  (Sort By Name)

		ls -lr (Sort By Reverse)



	See the all mode or permission and time details  of files ?
		
		ls -l

	See the Memory with bytes ?

		ls -l

	See the Memory with MB ?

		ls -ltrh


	see the count of ls ?
		
		ls -1 | wc -l

		Where
			-1 --> only one entry per line (-1a if you want hidden files too )
			wc --> wordcount
			-l --> count lines
