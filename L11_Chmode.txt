Syntax 

	chmod 777 name.txt



Permission of File 

	r - Read
	w - Write
	e - Executable


Permission Details 

	0 = 0      = nothing
	
	1 = 1      = executable(x)

	2 = 2      = write (w)

	3 = 2+1    = w + x

	4 = 4      = read (r)

	5 = 4+1    = r + x

	6 = 4+2    = r + w

	7 = 4+2+1  = r + w + x
